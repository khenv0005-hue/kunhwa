# 2) 소개 자료 (비점오염 통합 예측 시스템)

## 1. 문제 정의
- **현장의 어려움**: 비점오염 모니터링은 예보에 의존하여 출동을 결정하나, 기상청 단일 예보와 실제 강수의 괴리로 인해 '허위 출동'이 빈번함.
- **의사결정 복잡성**: KMA, GFS, ECMWF 등 여러 모델을 개별적으로 확인하고 판단하는 과정에서 시간이 소요되고 객관적인 기준이 부족함.
- **시각화 부재**: 수치 데이터만으로는 비구름의 이동 경로와 우리 사업지점의 선후 관계를 직관적으로 파악하기 어려움.

## 2. 해결 접근 (Solution)
- **4개 글로벌 모델 앙상블**: 한국, 미국, 유럽, 일본의 예보 데이터를 한 곳에 모아 예측의 신뢰도를 확보함.
- **다중 시간창 분석 엔진**: 1시간부터 72시간까지의 미래를 6개 구간(Short to Long-term)으로 쪼개어 각 시점별 대응 강도를 자동 계산함.
- **자체 역학적 보정 알고리즘**: 서해상 강우대의 이동 벡터와 지형(산맥)의 증폭 효과를 반영하여 격자 단위 정밀도를 높임.
- **통합 대시보드 UI**: 레이더, 수치 모델, GIS 경계, 유출 시뮬레이션을 원스롭으로 제공하여 의사결정 시간을 단축함.

## 3. 데모 화면 및 주요 흐름
| 주요 화면 | 기능 설명 |
| :--- | :--- |
| **통합 관제 지도** | Leaflet 기반 다크 테마 지도로 실시간 레이더 및 윈디(Windy) 레이어 표시 |
| **앙상블 비교 차트** | 4개 기상 모델의 시간별 강수량 예측치를 꺾은선 그래프로 동시 비교 |
| **의사결정 엔진 카드** | 강수 강도와 확률에 따라 6단계(즉시~미대응) 등급 및 대응 사유 자동 생성 |
| **GIS 통합 캔버스** | 사용자 유역 경계(SHP, GeoJSON) 드래그 앤 드롭 및 자동 매핑 |

## 4. 기대 효과
- **비용 절감**: 불필요한 허위 출동을 30% 이상 감소시켜 유류비 및 인건비 최적화.
- **대응력 향상**: 1~3시간 내 조기 예보(즉시 대응)를 통해 오염물질 유출 전 현장 도착률 제고.
- **데이터 기반 행정**: 주관적 판단이 아닌 객관적 데이터 수치에 근거한 보고 및 의사결정 체계 확립.

## 5. 한계 및 리스트
- **API 의존성**: 오픈소스 기상 API의 일일 호출 한도 초과 시 데이터 갱신 제한 가능성 (유료 API 전환으로 해결 가능).
- **예보 불확실성**: 국지성 호우와 같은 돌발 기상 상황에서는 모든 모델의 예측이 빗나갈 리스크 상존.

## 6. 확산 계획
- **모바일 앱 확장**: 현장 요원이 스마트폰으로 즉시 알림을 받을 수 있도록 PWA(Progressive Web App) 기술 적용 완료.
- **IoT 센서 연동**: 실시간 하천 수질/수위 센서 데이터를 대시보드에 결합하여 '예측'과 '실측'의 통합 모니터링 구현.

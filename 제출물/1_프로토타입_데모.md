# 1) 프로토타입 데모 시연 가이드

본 프로젝트의 결과물은 웹 기반 대시보드 형태의 의사결정 지원 시스템입니다. 아래 절차에 따라 시연을 진행할 수 있습니다.

## 🚀 실행 방법
1. **로컬 실행**: 
   - `frontend/index.html` 파일을 크롬/엣지 브라우저에서 실행합니다.
   - (권장) VSCode 의 `Live Server` 확장을 사용하여 실행 시 안정적인 로딩이 가능합니다.
2. **백엔드 연동 (선택사항)**:
   - `backend` 폴더에서 `pip install -r requirements.txt` 실행 후 `python main.py`를 실행하면 API 서버가 가동됩니다. (현재 프론트엔드는 호환성을 위해 직접 Open-Meteo API를 호출하도록 구성되어 있습니다.)

## 🎥 시연 시나리오 (3~5분 구성)

### [Step 1] 시스템 개요 및 접속 (30초)
- **화면 구성**: 다크 모드 기반의 프리미엄 대시보드 UI 설명. 좌측 지도, 우측 데이터 카드 레이아웃.
- **배경 설명**: "비점오염 모니터링을 위해 매번 기상청 레이더와 예보를 번갈아 확인하던 번거로움을 해결하기 위한 통합 솔루션"임을 강조.

### [Step 2] 실시간 레이더 및 기상 레이어 (1분)
- **KMA 레이더**: 좌측 레이어 컨트롤에서 'KMA 기상 레이더'를 활성화하여 현재 비구름의 이동 경로를 애니메이션으로 확인.
- **Windy 통합**: 'Global Windy' 레이어를 켜서 광범위한 기계학습 기반 비구름 예측 모델(ECMWF) 시각화.

### [Step 3] 다중 모델 기반 지점 분석 및 의사결정 (1.5분)
- **지점 선택**: 지도상의 특정 지점(유역 하구 등)을 클릭.
- **데이터 갱신**: [데이터 갱신] 버튼 클릭 시, **KMA(한국), GFS(미국), ECMWF(유럽), JMA(일본)** 4개 글로벌 모델의 72시간 예보 데이터를 실시간으로 가져와 앙상블 비교 차트 생성.
- **의사결정 엔진**: 하단의 **[다중 시간창 의사결정 가이드]** 확인. 1~72시간을 6개 구간으로 나누어 강수량/확률을 분석하고 [즉시 대응]~[미대응] 단계를 자동으로 판정하는 로직 시연.

### [Step 4] GIS 데이터 및 유출 시뮬레이션 (1분)
- **GIS 드래그 앤 드롭**: 준비된 GeoJSON이나 SHP(zip) 파일을 지도 위로 드래그하여 유역 경계가 즉시 매핑되는 기능 시연.
- **유출 입자 시뮬레이션**: 특정 지점 클릭 후 '유출 시뮬레이션' 실행 시, 지형에 따른 오염물질 이동 경로와 도달 시간 예측 애니메이션 시연.

---
> [!TIP]
> 시연 시 **'강수 확률이 모델별로 상이한 경우'**를 보여주며, "이러한 불확실성을 통합 시스템이 어떻게 분석하여 최적의 출동 시점을 제안하는지" 설명하면 효과적입니다.

# 비점오염 통합 예측 시스템 (하천 수질 모니터링 대시보드)

이 프로젝트는 전국 하천 유역 및 특정 지점을 대상으로 과거 관측 강수, 현재 레이더 영상, 미래 **4개 글로벌 기상 모델**(KMA, GFS, ECMWF, JMA) 예측 데이터를 통합 분석하여, 비점오염 현장 모니터링 "출동" 여부를 결정해주는 지능형 의사결정 시스템입니다.

단순 날씨 제공에서 벗어나, **다중 시간창(1~72시간) 분석 엔진**이 탑재되어 긴급성에 따른 6단계 대응 가이드를 제공합니다.

---

## 🚀 프로젝트 구조

```text
root/

│   └── requirements.txt    # 파이썬 의존성 목록
│
└── research/               # 기상 및 비점오염 분석 연구 자료
```

## 🌟 핵심 기능

1.  **실시간 기상 모델 앙상블 비교**:
    *   한국(KMA), 미국(GFS), 유럽(ECMWF), 일본(JMA)의 72시간 예측 데이터를 동시에 시각화하여 예측 불확실성을 최소화합니다.
2.  **다중 시간창 의사결정 엔진 (Decision Support)**:
    *   단기(1~3h)부터 장기(24~72h)까지 6개 시간창으로 구분하여 강수량 및 강수 확률을 분석합니다.
    *   **6단계 등급**: [즉시 대응], [대응 준비], [예의주시], [사전 계획], [장기 검토], [미대응]
3.  **KMA 실시간 레이더 연동**:
    *   RainViewer API를 통한 실시간 기상 레이더 에코 영상 재생 및 애니메이션 기능을 제공합니다.
4.  **GIS 데이터 시각화**:
    *   GeoJSON 및 Shapefile(.zip) 파일을 브라우저로 드래그 앤 드롭하여 사용자 정의 유역/경계를 즉시 지도상에 렌더링합니다.
5.  **글로벌 기상 레이어 (Windy)**:
    *   전 세계 수치 예보 모델(ECMWF) 실시간 바람장 및 비구름 레이어를 통합 제공합니다.

## 🛠️ 의사결정 판단 기준 (KMA/Multi-Model 기준)

| 대응 단계 | 시간 분석 구간 | 상세 판단 기준 (강수량 및 확률) |
| :--- | :--- | :--- |
| **🔴 즉시 대응** | 1~3시간 내 | 강수량 30mm 이상 또는 강수확률 85% 이상 |
| **🟠 대응 준비** | 3~6시간 내 | 강수량 20mm 이상 또는 강수확률 70% 이상 |
| **🟡 예의주시** | 6~12시간 내 | 강수량 10mm 이상 또는 강수확률 50% 이상 |
| **🔵 사전 계획** | 12~24시간 내 | 강수량 10mm 이상 또는 강수확률 50% 이상 |
| **🟣 장기 검토** | 24~72시간 내 | 강수량 10mm 이상 또는 강수확률 50% 이상 |
| **⚪ 미대응** | 전체 구간 | 위 모든 조건 미달 시 |

## 📦 실행 방법

### 1. 백엔드 API 서버 (Optional)
사용자가 백엔드를 통해 통합 데이터를 수신하고자 할 때 사용합니다.
```bash
cd backend
pip install -r requirements.txt
python main.py
```
*   API 서버: `http://localhost:8000`

### 2. 프론트엔드 대시보드
`frontend/index.html` 파일을 브라우저로 직접 열거나, VSCode `Live Server`를 통해 실행합니다.

# 비점오염 실시간 예측 AI 가이드

이곳은 **"기상 예보 연동 실시간 비점오염 예측 시스템"** 개발을 위한 메인 폴더입니다.
AI 모델을 처음 만드실 때 헷갈리지 않게 과정별로 폴더를 나누어 두었습니다.

## 📂 폴더 구성 안내
1. **01_원천데이터**: 측정소, 기상청, 국가공간정보포털 등에서 다운로드 받은 '원본 파일'을 그대로 보관하는 곳입니다. (절대 수정하지 마세요!)
2. **02_전처리데이터**: 원본 데이터를 AI가 학습하기 좋게 엑셀 하나로 깔끔하게 합치고 다듬은 결과물을 저장합니다.
3. **03_AI모델_실험**: 주피터 노트북(Jupyter) 코드나 오렌지3, 엑셀 AI 툴의 결과물, 모델 파일(.pkl, .h5 등)을 저장합니다.
4. **04_참고자료**: 관련 논문, 부서 내 보고서, 지침서 등을 모아두세요.

## 🚀 첫 번째 과제
`01_원천데이터` 폴더 안에 있는 각 하위 폴더들에 과거 데이터(엑셀, shapefile 등)를 모아주세요!
